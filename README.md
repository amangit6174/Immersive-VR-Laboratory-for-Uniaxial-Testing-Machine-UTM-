# Immersive VR Laboratory for Uniaxial Testing Machine (UTM)

An immersive Virtual Reality laboratory developed using **Unity**, **Blender**, and **C#** to simulate the working of a **Uniaxial Testing Machine (UTM)** for engineering education. The project enables students to perform virtual laboratory experiments in an interactive and realistic environment using Meta Quest devices.

---

## Project Overview

This project aims to improve practical engineering education by providing a virtual laboratory where users can interact with a digital twin of a Uniaxial Testing Machine. It minimizes dependence on physical laboratory infrastructure while enhancing accessibility and learning through immersive Virtual Reality.

---

## Features

- Immersive Virtual Reality laboratory experience
- Interactive Uniaxial Testing Machine (UTM) simulation
- Realistic 3D models created in Blender
- User interaction and machine functionality implemented using C#
- XR Interaction Toolkit integration
- Compatible with Meta Quest devices
- Interactive engineering learning environment
